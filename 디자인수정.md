# STUDIYA 프로젝트 네오브루탈리즘 전환 수정 계획

## 📋 프로젝트 개요
- **프로젝트명**: STUDIYA 수학 튜터 MVP
- **현재 상태**: Neo-Brutalism with Soft Touches (브루탈리즘 10% 적용)
- **목표**: 진정한 Z세대 네오브루탈리즘 웹사이트로 전환
- **타겟**: 중학생 (12-15세) + 학부모
- **핵심 메시지**: "투박하지만 솔직하고 강력한" 학습 경험

## 🎨 네오브루탈리즘 핵심 원칙

### 1. 색상 철학
```
기존: 부드럽고 따뜻한 색상 (베이지, 연핑크)
변경: 강렬하고 대담한 색상 (순수 컬러, 높은 대비)
```

### 2. 타이포그래피 철학
```
기존: 세련된 Inter 폰트, 적당한 굵기
변경: 투박한 Arial Black, 최대 굵기, 공격적 스타일
```

### 3. 레이아웃 철학
```
기존: 완벽한 정렬, 깔끔한 격자
변경: 의도적 비대칭, 살짝 기울어진 요소들
```

### 4. 인터랙션 철학
```
기존: 부드러운 애니메이션, 은은한 피드백
변경: 즉각적 반응, 강렬한 시각적 피드백
```

```typescript
// 파일 위치: src/components/ui/Button.tsx
interface ButtonProps {
  variant: 'primary' | 'secondary' | 'danger' | 'success';
  size: 'sm' | 'md' | 'lg';
  rotation?: 'none' | 'slight' | 'mild' | 'bold';
  children: React.ReactNode;
  className?: string;
  onClick?: () => void;
  disabled?: boolean;
}

const Button = ({ 
  variant = 'primary', 
  size = 'md', 
  rotation = 'slight',
  children, 
  className = '',
  ...props 
}) => {
  const baseClasses = [
    'font-black uppercase tracking-wide',
    'border-4 border-black',
    'transition-all duration-75 ease-out',
    'cursor-pointer inline-block text-center',
    'hover:translate-x-1 hover:translate-y-1',
    'hover:shadow-[4px_4px_0px_0px_#000]',
    'active:translate-x-2 active:translate-y-2',
    'active:shadow-[2px_2px_0px_0px_#000]'
  ];

  const variantClasses = {
    primary: 'bg-yellow-400 text-black hover:bg-orange-500',
    secondary: 'bg-white text-black hover:bg-pink-500 hover:text-white',
    danger: 'bg-red-500 text-white hover:bg-red-600',
    success: 'bg-green-400 text-black hover:bg-green-500'
  };

  const sizeClasses = {
    sm: 'px-4 py-2 text-sm shadow-[4px_4px_0px_0px_#000]',
    md: 'px-6 py-3 text-base shadow-[6px_6px_0px_0px_#000]',
    lg: 'px-8 py-4 text-lg shadow-[8px_8px_0px_0px_#000]'
  };

  const rotationClasses = {
    none: 'rotate-0',
    slight: 'rotate-1 hover:rotate-0',
    mild: 'rotate-2 hover:rotate-0',
    bold: 'rotate-3 hover:rotate-0'
  };

  return (
    <button
      className={[
        ...baseClasses,
        variantClasses[variant],
        sizeClasses[size],
        rotationClasses[rotation],
        className
      ].join(' ')}
      {...props}
    >
      {children}
    </button>
  );
};
```

```typescript
// 파일 위치: src/components/features/AnimatedQuizTimer.tsx
const BrutalQuizTimer = ({ timeLeft, totalTime = 45 }) => {
  const percentage = (timeLeft / totalTime) * 100;
  
  // 색상 단계적 변화
  const getTimerColor = () => {
    if (percentage > 60) return 'bg-blue-500';
    if (percentage > 30) return 'bg-orange-500';
    return 'bg-red-500';
  };

  const getTimerBorderColor = () => {
    if (percentage > 60) return 'border-blue-600';
    if (percentage > 30) return 'border-orange-600';
    return 'border-red-600';
  };

  return (
    <div className="w-full mb-6">
      {/* 타이머 바 컨테이너 */}
      <div className="w-full h-12 border-4 border-black bg-white relative overflow-hidden">
        {/* 진행 바 */}
        <div 
          className={`h-full transition-all duration-1000 ${getTimerColor()}`}
          style={{ width: `${percentage}%` }}
        />
        
        {/* 시간 텍스트 (중앙) */}
        <div className="absolute inset-0 flex items-center justify-center">
          <span className="font-black text-xl text-black bg-white px-2 border-2 border-black">
            {timeLeft}초
          </span>
        </div>
        
        {/* 픽셀아트 스타일 테두리 강조 */}
        <div className={`absolute inset-0 border-2 ${getTimerBorderColor()}`} />
      </div>
      
      {/* 상태 메시지 */}
      <div className="text-center mt-2">
        <span className={`
          font-black text-sm px-3 py-1 border-2 border-black
          ${percentage > 60 ? 'bg-blue-100' : ''}
          ${percentage <= 60 && percentage > 30 ? 'bg-orange-100' : ''}
          ${percentage <= 30 ? 'bg-red-100' : ''}
        `}>
          {percentage > 60 ? '충분한 시간' : percentage > 30 ? '서둘러요!' : '시간이 얼마 없어요!'}
        </span>
      </div>
    </div>
  );
};
```

```typescript
                href={item.href}
                className={`
                  font-bold text-black px-4 py-2 border-3 border-black
                  hover:bg-pink-400 transition-all duration-75

                  hover:rotate-0 hover:shadow-[2px_2px_0px_0px_#000]
                `}
              >
                {item.name}
              </a>
            ))}
          </nav>
          
          {/* 모바일 햄버거 메뉴 */}
          <button 
            className="md:hidden border-4 border-black bg-white p-2
                      hover:bg-red-400 transition-all duration-75"
            onClick={() => setIsMenuOpen(!isMenuOpen)}
          >
            <div className="w-6 h-6 flex flex-col justify-center space-y-1">
              <div className="w-full h-1 bg-black" />
              <div className="w-full h-1 bg-black" />
              <div className="w-full h-1 bg-black" />
            </div>
          </button>
        </div>
      </div>
      
      {/* 모바일 메뉴 */}
      {isMenuOpen && (
        <div className="md:hidden bg-white border-t-4 border-black">
          {/* 모바일 메뉴 아이템들 */}
        </div>
      )}
    </header>
  );
};
```

```typescript
  ];

  const variantClasses = {

    info: 'bg-blue-400'
```
